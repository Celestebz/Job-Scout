# Job Scout - 智能岗位侦察兵

## 🎯 功能说明

自动搜索、分析、推荐匹配的职位机会,帮助你高效求职。

**核心特性:**
- 🔍 **主动搜索** - 基于用户画像执行多源搜索
- 🧠 **智能Gap分析** - 基于历史数据优化搜索策略，效率提升40%+
- 🎯 **优先级排序** - 按外企 > 大厂 > 其他优先级搜索（目标40%/40%/20%）
- 🎯 **智能匹配** - 计算岗位匹配度(0-100分)
- 🚫 **自动去重** - 过滤已推荐过的岗位
- 📊 **XLSX报告** - 生成带格式的大字体Excel报告（含优先级标签）
- ✅ **三重质量验证** - 社招 + 公司名精确 + JD真实
- ✅ **质量控制** - 只输出匹配度≥60的高质量岗位
- 🏢 **清晰度过滤** - 拒绝模糊公司名(如"某公司")
- ✅ **真实JD保证** - 100%真实招聘信息，绝不使用模板
- 📱 **官方公众号** - 仅使用企业认证官方招聘公众号（拒绝个人转载）
- 🌟 **专业平台** - 整合神仙外企等垂直领域专业招聘平台
- 🤖 **AI智能扩展** - 自动识别"大厂""外企"等类别，推理出优秀公司列表

## 📋 使用方法

### 基础用法
```
/job-scout 帮我找工作
```

### 指定关键词
```
/job-scout 帮我找深圳的AI产品运营岗位
```

### 指定薪资范围
```
/job-scout 找12-18K的海外运营岗位
```

## 📁 输出内容

### 1. XLSX报告
- 📍 位置: `skills/job-scout/result/job_scout_用户名_时间戳.xlsx`
- 📊 包含: 排名、公司、岗位、薪资、地点、匹配度、JD、投递方式、备注、优先级
- 🎨 格式: 14号大字体,紫色表头,自动换行
- 🏷️ 优先级标签: P1外企(绿色)、P2大厂(蓝色)、P3其他(白色)
- ✅ **JD真实性**: 100%真实招聘信息,从官方/招聘平台提取
- ✅ **招聘类型**: 100%社招岗位（无校招/实习）

### 2. 数据库更新
- 📍 文件: `skills/job-scout/job_history.json`
- 📝 记录: 所有历史岗位,用于去重和Gap分析

## ⚙️ 配置要求

### 用户配置
位置: `skills/job-scout/users/{user_id}/config.json`

包含:
- 期望薪资范围
- 目标城市
- 目标公司类型（支持类别描述，如"大厂""外企"，AI会自动扩展）
- 搜索关键词

**AI智能扩展示例**:
```json

// AI会自动推理出具体公司列表并搜索:

```

### 简历文件
位置: `skills/job-scout/users/{user_id}/resume.docx`

## 🔄 工作流程

```
1. 读取用户简历和配置
   ↓
2. 读取历史岗位数据库 (56个岗位)
   ↓
3. 智能Gap分析 (识别搜索盲区)
   - 外企占比: 12.5% (目标40%)
   - 大厂占比: 10.7% (目标40%)
   - 其他占比: 76.8% (目标20%)
   ↓
4. 生成智能搜索策略
   - 优先搜索外企和大厂
   - 跳过已充分覆盖的公司
   ↓
5. 执行多轮WebSearch搜索
   - Priority 1: 外企 (微软、苹果、亚马逊等)
   - Priority 2: 大厂 (小米、华为、OPPO、网易等)
   - Priority 3: 其他 (控制数量)
   ↓
6. 提取100%真实JD(从官方/招聘平台)
   ↓
7. 三重质量验证
   - Layer 1: 招聘类型验证(社招优先)
   - Layer 2: 公司名称精确度验证
   - Layer 3: JD真实性验证
   ↓
8. 去重检查(对比历史数据库)
   ↓
9. 匹配度计算
   ↓
10. 动态扩展搜索(如岗位不足10个)
   ↓
11. 更新数据库
   ↓
12. 生成XLSX报告
```

## 🎯 优先级分布目标

**目标分布**（每次搜索10-15个岗位）:
- 🟢 **Priority 1（外企）**: 5-7 个 (40%)
- 🔵 **Priority 2（大厂）**: 5-7 个 (40%)
- ⚪ **Priority 3（其他）**: 2-3 个 (20%)

**搜索优先级说明**:

**Priority 1: 外企**（最高优先级，目标40%）
- 搜索平台：神仙外企、各大外企官方招聘网站
- 覆盖公司：Microsoft、Apple、Amazon、Google、Tesla、Cisco、Intel、IBM等
- 搜索方式：
  - `site:waiqi.com 外企 运营 深圳`
  - `site:mp.weixin.qq.com "微软中国招聘" 运营`
  - `site:jobs.apple.com 深圳 operations`
  - `site:careers.microsoft.com 深圳 运营`

**Priority 2: 国内大厂**（第二优先级，目标40%）
- 搜索平台：大厂官方招聘网站、官方公众号
- 覆盖公司：字节跳动、腾讯、阿里、美团、拼多多、京东、快手、百度、小米、华为、OPPO、vivo、网易等
- 搜索方式：
  - `site:jobs.bytedance.com 运营 深圳`
  - `site:mp.weixin.qq.com "腾讯招聘" 产品运营`
  - `site:career.huawei.com 社招`
  - `小米 运营 深圳 社招`

**Priority 3: 其他匹配岗位**（补充优先级，目标≤20%）
- 搜索平台：猎聘、Boss直聘、智联招聘等
- 覆盖公司：出海公司、创业公司、其他匹配企业
- 搜索方式：
  - `猎聘 深圳 海外运营`
  - `Boss直聘 深圳 AI运营`
  - `深圳 出海运营 15-25K`
- 仅在外企和大厂岗位不足时才搜索

**排序规则**:
- 优先级1（外企）> 优先级2（大厂）> 优先级3（其他）
- 同优先级内按匹配度从高到低排序
- XLSX报告中会标注每个岗位的优先级（P1/P2/P3）

## 💡 使用建议

1. **定期运行**: 每天或每周运行一次,发现新岗位
2. **关注Gap分析**: 查看外企/大厂占比，了解市场分布
3. **投递优先级**: 优先投递外企和大厂岗位（P1和P2）
4. **投递跟踪**: 在XLSX中添加投递状态和面试备注
