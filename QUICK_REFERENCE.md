- **Priority 1**: 外企（微软、苹果、亚马逊等）
- **Priority 2**: 大厂（字节、腾讯、阿里等）

- 匹配度 ≥ 60 分
- 公司名必须清晰
- JD必须真实

## 🎯 配置说明

### config.json 结构
```json
{
  "search_preferences": {
    "location": "深圳",
    "target_companies": ["字节跳动", "腾讯"],
    "company_types": ["大厂", "外企", "出海公司"],
    "industries": ["出海", "AI", "AIGC"],
    "job_types": ["运营", "产品运营", "海外运营"]
  },
  "salary_expectation": {
    "min": 12000,
    "target": 15000,
    "max": 18000
  },
  "matching_config": {
    "quality_threshold": 70,
    "fallback_threshold": 60
  }
}
```

### 关键配置项
| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `location` | 工作地点 | 深圳 |
| `company_types` | 公司类型 | 大厂/外企/出海 |
| `salary.min` | 最低薪资 | 12000 |
| `quality_threshold` | 质量阈值 | 70 |

## 🛠️ 常见问题

### Q1: 为什么有些公司被跳过？
**A**: 已充分覆盖的公司（≥5个岗位）会自动跳过，避免重复搜索。

### Q2: 如何提高外企岗位比例？
**A**: 
1. Gap分析会自动识别外企不足
2. 自动优先搜索外企平台（神仙外企、外企官网）
3. 调整 `config.json` 中的 `company_types`

### Q3: 匹配度怎么计算的？
**A**: 
- 直接匹配（技能匹配）：50%
- 可迁移经验：30%
- 薪资匹配：10%
- 公司优先级：10%

### Q4: 如何自定义搜索策略？
**A**: 编辑 `config.json`：
```json
{
  "search_preferences": {
    "target_companies": ["小米", "华为"],  // 新增目标公司
    "excluded_keywords": ["销售", "客服"]   // 排除关键词
  }
}
```
